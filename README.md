# PPLBandage
[![Made for pepeland](https://andcool.ru/static/badges/made-for-ppl.svg)](https://pepeland.net)  
**Исходный код сайта с повязками.**

## Запуск

1. Установка зависимостей  
    `npm install`
2. Сборка исходников  
    `npm run build`
3. Запуск собранного сайта  
   `npm run start`
